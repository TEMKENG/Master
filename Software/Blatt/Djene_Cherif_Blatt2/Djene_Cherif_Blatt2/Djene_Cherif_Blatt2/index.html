Wählen Sie auf der linken Seite Ihr Zeichwerkzeug aus. Haben Sie eines ausgewählt, können Sie mit der Maus die entsprechenden Figuren zeichen. Typischerweise, indem Sie die Maus drücken, dann mit gedrückter Maustaste die Form bestimmen, und dann anschließend die Maustaste loslassen.

Wenn Sie den Menüpunkt "Selektion"" auswählt, können Sie die Graphikobjekte durch Klick selektionieren. Selektionierte Objekte erhalten kleine Rechtecke an Ihren Enden.
Falls mehrere Graphikobjekte in Frage kommt, können Sie diese durch mehrmalige Klicken durchschalten. Und Wenn Sie mehrere Objekte auf einmal selektionieren möchten, halten Sie beim Klicken die STRG-Taste gedrückt.
Mit dem Button "Selektion löschen" können Sie selektierte Objekte aus der Zeichenfläche löschen.
